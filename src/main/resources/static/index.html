一键部署

引导式部署向导，串联安装目录、端口、数据库、WAR 包与 YML 配置，适合首次环境落地。

  • 支持 Windows / Linux
  • 内置多环境配置模板
  • 实时部署日志与服务检测

模拟数据管理

基于配置模板连接目标库，支持从业务库导出模拟数据（DDL + XML），后续将补充反向导入能力，覆盖常见联调与回归场景。

  • 模拟数据导出：从达梦业务库按 Schema / 对象清单导出 DDL + XML
  • 模拟数据导入：将导出的 ZIP 一键导入目标达梦环境
  • 统一 ZIP 包管理，便于跨环境备份与流转
